# 无尽的拉格朗日公告邮件撰写助手

一个专为《无尽的拉格朗日》游戏设计的多色文本编辑器，用于创建和编辑带有颜色标记的公告邮件内容。

## 功能特性

- **实时彩色预览**：在编辑器中直接预览文本的颜色效果
- **颜色选择**：右键选中文本可打开颜色选择面板，自定义文本颜色
- **智能坐标高亮**：自动识别并高亮显示坐标格式（如 (123,456)）
- **导入导出功能**：支持拉格朗日格式的文本导入和导出
- **撤销重做**：支持 Ctrl+Z 撤销和 Ctrl+Y 重做功能
- **文本长度检查**：实时显示当前文本长度和限制状态

## 使用说明

1. 上方文本框为彩色效果预览框，可直接编辑和设置文本颜色
2. 下方文本框为原始内容输出框，显示可在邮件中直接使用的格式化内容
3. 在上方文本框中框选字符后按右键可打开颜色选择面板
4. 支持使用 "Ctrl-Z" 撤回和 "Ctrl-Y" 重做操作

## 拉格朗日聊天修饰符号

### 普通玩法

#### 预设颜色代码
| 颜色 | 快捷设置代码 | 实际16位颜色代码 |
|------|-------------|-----------------|
| 蓝色 | #B | #0000FF |
| 金色 | #D | #FFD700 |
| 绿色 | #G | #00FF00 |
| 黑色 | #K | #000000 |
| 橙色 | #O | #FFA500 |
| 粉色 | #P | #FFC0CB |
| 红色 | #R | #FF0000 |
| 紫色 | #U | #800080 |
| 白色 | #W | #FFFFFF |
| 黄色 | #Y | #FFFF00 |

> 注：拉格朗日邮件文本框背景色为 #0E0E0E，聊天框背景色为 #050505

#### 高级颜色设置
使用 `#c` + HEX颜色代码的格式可以实现更多颜色的显示。

#### 文本控制符号
- `#r` - 换行
- `#l` (小写的L) - 使用上一个颜色（Last Color）
- `#n` - 清除颜色（No Color），实际显示为灰色
- `#E` - 下划线（只能在聊天框使用，邮件中无效）
- `#e` - 下划线的停止符

**示例**：`#E测试#e测试` → <u>测试</u>测试
